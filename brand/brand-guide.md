# PixelForge — Брендбук

## 1. Цветовая палитра

### Основные цвета

| Роль | Цвет | HEX | RGB | Применение |
|---|---|---|---|---|
| Основной | Тёмно-синий (Navy) | #1A2B4A | 26, 43, 74 | Заголовки, текст логотипа, основные элементы интерфейса |
| Акцентный | Светло-синий (Azure) | #4D9EFF | 77, 158, 255 | Акценты, кнопки, ссылки, декоративные элементы, пиксели наковальни |
| Фоновый | Белый | #FFFFFF | 255, 255, 255 | Фон, пространство, "воздух" в композиции |

### Дополнительные цвета

| Роль | HEX | Применение |
|---|---|---|
| Тёмный фон | #0D1B2A | Тёмная тема, футер, инвертированные блоки |
| Серый текст | #6B7B8D | Вторичный текст, подписи, метаданные |
| Светлый фон | #F4F7FA | Альтернативный фон секций, карточки |
| Успех | #2ECC71 | Уведомления об успехе, положительные метрики |
| Ошибка | #E74C3C | Ошибки, предупреждения |

### Правила использования

- Основной цвет #1A2B4A занимает не менее 60% цветовой площади
- Акцентный #4D9EFF — не более 20%, используется точечно
- Белый — не менее 20%, обеспечивает "воздух"
- Никогда не использовать акцентный цвет для больших текстовых блоков
- В монохромной печати основной цвет = чёрный, акцентный = 50% серый

## 2. Типографика

### Шрифтовая система

| Роль | Шрифт | Начертание | Размер (web) |
|---|---|---|---|
| Заголовки H1 | Segoe UI / Inter | Bold (700) | 36-48px |
| Заголовки H2 | Segoe UI / Inter | SemiBold (600) | 24-32px |
| Основной текст | Segoe UI / Inter | Regular (400) | 16-18px |
| Подписи, метаданные | Segoe UI / Inter | Light (300) | 12-14px |
| Код, технические данные | JetBrains Mono / Fira Code | Regular (400) | 14-16px |

### Логотип: текстовая часть

- "Pixel" — начертание Light (300)
- "Forge" — начертание Bold (700)
- Контраст начертаний обязателен при любом воспроизведении
- Минимальный размер: 24px по высоте (web), 8mm (печать)

### Правила

- Межстрочный интервал: 1.5-1.6 для основного текста
- Максимальная длина строки: 75 символов
- Выравнивание: по левому краю (не по ширине)

## 3. Логотип

### Описание

Комбинированный знак: графический элемент (пиксельная наковальня) + текстовая часть (PixelForge).

### Варианты использования

| Вариант | Применение |
|---|---|
| Горизонтальный (полный) | Шапка сайта, документы, презентации |
| Графический элемент (без текста) | Фавикон, иконка приложения, водяной знак |
| Монохромный (тёмный на светлом) | Печатные материалы, штампы |
| Инвертированный (светлый на тёмном) | Тёмный фон, футер |

### Охранное поле

Минимальное расстояние от логотипа до других элементов = высота буквы "P" в слове "Pixel".

### Запреты

- Не растягивать и не сжимать непропорционально
- Не поворачивать
- Не менять цвета на нефирменные
- Не размещать на пёстром или контрастном фоне без подложки
- Не добавлять тени, обводки, свечения

**Правильно:** "Расскажите о вашем проекте — обсудим задачи и предложим решение."

## 5. Фирменные элементы

### Пиксельная сетка

Фирменный паттерн — сетка из квадратных блоков (8x8px, 16x16px или 32x32px) с чередованием основного и акцентного цветов. Используется как фоновый декоративный элемент.

### Кодовые символы

Символы `</>` используются как декоративный элемент в заголовках, разделителях и иконках. Символизируют основную деятельность — веб-разработку.

### Иконографика

Стиль иконок: линейные (outline), толщина линии 2px, цвет — основной (#1A2B4A) или акцентный (#4D9EFF). Скругление углов: 2px.
