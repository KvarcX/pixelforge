# BoostLab — Брендбук

## 1. Цветовая палитра

### Основные цвета

| Роль | Цвет | HEX | RGB | Применение |
|---|---|---|---|---|
| Основной | Фиолетовый (Violet) | #7C3AED | 124, 58, 237 | Заголовки, контур логотипа (колба), основные элементы интерфейса, бренд-акценты |
| Акцентный | Изумрудный (Emerald) | #10B981 | 16, 185, 129 | Метрики роста, графики, кнопки CTA, положительные показатели, «жидкость» в колбе логотипа |
| Фоновый | Белый | #FFFFFF | 255, 255, 255 | Фон, «воздух» композиции, чистота лабораторной среды |

### Дополнительные цвета

| Роль | HEX | Применение |
|---|---|---|
| Тёмный графит | #0F172A | Тёмная тема, футер, инвертированные блоки, основной текст |
| Серый текст | #64748B | Вторичный текст, подписи, метаданные |
| Светлый фон | #F8FAFC | Альтернативный фон секций, карточки, панель дашборда |
| Успех / рост | #10B981 | Дубликат акцента: уведомления, положительные метрики (+% в кейсах) |
| Предупреждение | #F59E0B | Падение метрик, требующие внимания показатели |
| Ошибка / падение | #EF4444 | Критические метрики, отрицательные показатели, ошибки |

### Правила использования

- Основной цвет #7C3AED занимает не более 40% цветовой площади (контрастный доминант, не перегружать)
- Акцентный #10B981 используется исключительно для «растущих» показателей и позитивных CTA — не для декора
- Белый — не менее 40%, лаборатория требует «воздуха»
- В монохромной печати основной цвет = 80% чёрный, акцентный = 40% серый
- Никогда не ставить фиолетовый и изумрудный вплотную друг к другу без белого зазора — возникает оптическая вибрация

## 2. Типографика

### Шрифтовая система

| Роль | Шрифт | Начертание | Размер (web) |
|---|---|---|---|
| Заголовки H1 | Inter / Segoe UI | Bold (700) | 36-48px |
| Заголовки H2 | Inter / Segoe UI | SemiBold (600) | 24-32px |
| Основной текст | Inter / Segoe UI | Regular (400) | 16-18px |
| Подписи, метаданные | Inter / Segoe UI | Light (300) | 12-14px |
| Код, числа метрик | JetBrains Mono / Fira Code | Regular / Medium | 14-16px |

### Логотип: текстовая часть

- **«Boost»** — начертание Bold (700): доминанта, действие, результат
- **«Lab»** — начертание Light (300), цвет #7C3AED: тонкая точность, метод, подпись к действию
- Контраст начертаний обязателен при любом воспроизведении
- Минимальный размер: 24px по высоте (web), 8mm (печать)

### Правила

- Межстрочный интервал: 1.5-1.6 для основного текста
- Максимальная длина строки: 75 символов
- Выравнивание: по левому краю (не по ширине)
- Числа метрик в тексте — моноширинным шрифтом, чтобы цифры «считывались как данные»

## 3. Логотип

### Описание

Комбинированный знак: графический элемент (лабораторная колба с восходящим графиком) + текстовая часть (BoostLab).

### Варианты использования

| Вариант | Применение |
|---|---|
| Горизонтальный (полный) | Шапка сайта, документы, презентации, коммерческие предложения |
| Графический элемент (без текста) | Фавикон, иконка приложения, водяной знак, лоадер |
| Монохромный (тёмный на светлом) | Печатные материалы, штампы, договоры |
| Инвертированный (светлый на тёмном) | Тёмный фон, футер, тёмная тема сайта |

### Охранное поле

Минимальное расстояние от логотипа до других элементов = высота буквы «B» в слове «Boost».

### Запреты

- Не растягивать и не сжимать непропорционально
- Не поворачивать (колба должна стоять вертикально — она лабораторный прибор, а не элемент декора)
- Не менять цвета на нефирменные; не инвертировать violet и emerald местами
- Не размещать на пёстром или контрастном фоне без белой/графитовой подложки
- Не добавлять тени, обводки, свечения; стрелка-график уже является финальным акцентом

## 4. Тон голоса (Tone of Voice)

### Характеристика

| Параметр | Наш голос | Не наш голос |
|---|---|---|
| Стиль | Аналитический, конкретный, сдержанно-уверенный | Канцелярит; «продающие» восклицания |
| Уверенность | Уверенный через цифры | Самовосхваляющий, «лучшие на рынке» |
| Сложность | Простой, но не упрощённый | Маркетинговый жаргон, buzzwords без содержания |
| Эмоции | Холодная ясность + внимание к клиенту | Восторженность, обилие восклицательных знаков |
| Юмор | Лёгкая ирония над «красивостью ради красивости» уместна | Мемы, сленг, панибратство |

### Примеры

**Правильно:** «Сократили стоимость заявки с 1200 до 480 ₽ за 6 недель.»
**Неправильно:** «Создаём ЛУЧШИЕ сайты в МИРЕ!!!»

**Правильно:** «Расскажите о задаче — снимем базовые метрики и сформулируем гипотезы роста.»
**Неправильно:** «Оставьте заявку и получите БЕСПЛАТНЫЙ аудит прямо СЕЙЧАС!»

### Ключевые формулировки бренда

| Концепция | Формулировка |
|---|---|
| Слоган | Каждая метрика работает на вас |
| Миссия | Превращать сайт из «витрины» в измеряемый инструмент роста бизнеса |
| Позиционирование | Лаборатория роста: гипотезы, эксперименты, цифры вместо прилагательных |
| Обещание | Любое решение на сайте подкреплено цифрой — или оно ещё не принято |

## 5. Фирменные элементы

### Сетка метрик

Фирменный паттерн — прямоугольная сетка с вложенными столбиками разной высоты (как мини-графики). Используется как фоновый декоративный элемент в шапках секций и разделителях. Базовый модуль: 8×8px; высоты столбиков — 1, 2, 3, 5, 8 модулей (последовательность Фибоначчи, «кривая роста»).

### Символы данных

Парные фигурные скобки `{ }` используются как декоративный элемент в заголовках, кнопках и иконках. Символизируют данные, JSON-контекст, «поле измерения» — в противовес классическим HTML-тегам, потому что BoostLab работает не с разметкой, а с данными.

### Иконографика

Стиль иконок: линейные (outline), толщина линии 2px, цвет — основной (#7C3AED) или акцентный (#10B981). Скругление углов: 2px. Для «положительных» иконок (рост, успех, подтверждение) — изумрудный, для навигационных и нейтральных — фиолетовый.

### Анимация и движение

- Стрелки графиков и индикаторы роста анимируются «снизу вверх» (direction: up). Никогда — сверху вниз.
- Продолжительность анимаций: 200-400ms, функция `ease-out` — имитация естественного замедления.
- Hover-эффекты на метриках допускают «подсветку» изумрудным — как включение прибора.
