# PixelForge

**Digital Craft Studio** — веб-студия, специализирующаяся на создании современных веб-приложений, корпоративных сайтов и цифровых продуктов.

## О нас

PixelForge — это "цифровая кузница", где из пикселей выковываются надёжные веб-продукты. Мы объединяем ремесленный подход к качеству с современными технологиями разработки.

### Направления

- Разработка адаптивных веб-сайтов и лендингов
- UI/UX дизайн
- Веб-приложения на заказ (SPA, PWA, интернет-магазины)
- Поддержка и развитие digital-продуктов

### Наши ценности

- **Честность.** Говорим правду до старта, а не после оплаты.
- **Детальность.** Каждый пиксель проверен на каждом устройстве.
- **Результат.** Создаём инструменты, которые приносят деньги.
